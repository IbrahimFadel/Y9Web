Explanation & Justification

We are living in the age of data. Our data is being tracked everyday with every google search, and every video we watch. Yet with so much data being collected, we still sometimes find it difficult to truly understand it. With so many organizations like the 'World Health Organization' presenting this data in it's raw, tabular form, we can then transform it into a beautifilly visualized representation using plain HTML, CSS, and Javascript. By downloading the raw data from the WHO's website into JSON format, we can then process this data and present it on the HTML5 canvas element. This provides a more readable, understandable interface from which anybody can understand the given data. The problem with the way data is presented is that we are only looking at numbers. Humans were not built to process numbers. We were built to take in information with our five senses. When looking at these numbers, it's difficult to truly understand the issues, and trends presented in the data. By representing this tabular data in the canvas element, we can help people better understand the data they are looking at, making it easy for real progress to be made!
